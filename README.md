# customer_behavior_analysis
Data Analytics project showcasing customer behavior analysis using MySQL, Python and Power BI


Here’s a clean, professional, recruiter-friendly README you can directly use 👇

## Overview

This project focuses on end-to-end data analytics, starting from raw data processing to generating business insights through visualization. It demonstrates practical skills in data cleaning, exploratory data analysis (EDA), SQL querying, and dashboard creation.

The workflow includes Python for data handling, MySQL for querying structured data, and Power BI for building an interactive dashboard.

##  Dataset

* The dataset contains structured data related to customer/business operations.
* It includes multiple features such as categorical and numerical variables.
* Data was preprocessed to handle missing values, duplicates, and inconsistencies.

## Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* MySQL(for data storage and querying)
* Power BI (for dashboard and visualization)
* Jupyter Notebook / VS Code


##  Project Workflow

### 1. Data Loading

* Imported dataset into Python using Pandas
* Performed initial inspection using `.head()`, `.info()`, `.describe()`

### 2. Exploratory Data Analysis (EDA)

* Analyzed data distributions and relationships
* Identified trends, patterns, and anomalies
* Used visualizations like histograms, bar charts, and heatmaps

### 3. Data Cleaning

* Handled missing values
* Removed duplicates
* Corrected data types
* Standardized inconsistent values

### 4. SQL Analysis (MySQL)

* Loaded cleaned data into MySQL database
* Wrote SQL queries for insights such as:

  * Aggregations (SUM, AVG, COUNT)
  * Grouping and filtering
  * Joins and subqueries
* Extracted meaningful business insights

### 5. Dashboard Creation (Power BI)

* Connected Power BI to MySQL / dataset
* Built interactive dashboard with:

  * KPI cards
  * Charts and graphs
  * Filters and slicers
* Designed for easy interpretation of insights


##  Dashboard

The Power BI dashboard provides a visual summary of key insights:

* Key performance indicators (KPIs)
* Trend analysis
* Category-wise comparisons
* Interactive filters for deeper exploration

##  Results & Insights

* Identified key trends and patterns in the dataset
* Highlighted important business metrics
* Improved data quality through preprocessing
* Enabled data-driven decision making through visual insights

##  Conclusion

This project demonstrates a complete data analytics pipeline, combining programming, database management, and visualization skills to extract meaningful insights from raw data.
